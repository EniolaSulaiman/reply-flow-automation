# ReplyFlow AI — AI Customer Support Automation Landing Page

A high-converting, SEO-optimized SaaS landing page for **ReplyFlow AI**, an AI-powered customer support automation tool built for Shopify and WooCommerce store owners. Built as a portfolio project to demonstrate modern frontend development and UI/UX design skills.

## 📸 Preview

> A fully responsive, single-page marketing site featuring animated SVG illustrations, a working pricing toggle, smooth scroll animations, and a mobile hamburger menu.

---

## ✨ Features

- **Ultra-modern SaaS aesthetic** — dark/light alternating sections, teal accent, generous whitespace
- **Fully responsive** — mobile, tablet, and desktop layouts with a hamburger nav on small screens
- **Custom inline SVGs** — automation flow diagram, analytics dashboard, chat UI, and integration map — no external icon libraries
- **Animated hero dashboard** — floating mockup with live chat simulation
- **Working pricing toggle** — switches between monthly and annual billing with JavaScript
- **Smooth scroll animations** — fade-up reveals triggered by IntersectionObserver
- **SEO-optimised structure** — semantic HTML5, proper H1→H3 hierarchy, meta tags, and FAQ schema markup
- **Accessibility** — ARIA labels, roles, and descriptive alt text throughout
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript, no frameworks or build tools

---

## 🗂️ Page Sections

1. **Hero** — H1 headline, animated dashboard mockup, trust logo row, dual CTAs
2. **Problem** — 6-card grid highlighting pain points for e-commerce support teams
3. **Solution** — Feature overview with an animated automation flow SVG
4. **Features** — Four alternating layout sections with supporting SVG illustrations
5. **Social Proof** — Testimonials with specific metrics and star ratings
6. **How It Works** — 3-step horizontal flow with connected step indicators
7. **Pricing** — 3-tier table with monthly/annual toggle and a money-back guarantee badge
8. **FAQ** — Schema-ready structured Q&A in a 2-column grid
9. **Final CTA** — Closing conversion section with dark background and radial gradient

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
